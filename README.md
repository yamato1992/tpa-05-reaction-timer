# タスク：Make two (2) simultaneous active cells.

![img](https://i.imgur.com/WhHXTwsm.png)

* 反応タイマーゲームは"active cell"が同時に二つ現れるように治してください。
* クリックする順番は関係ありません。
* 一つ目のセルをクリックすれば、スタート時間からクリックする瞬間までの時間を測って、コンソールで表示します。
* 二つ目のセルをクリックすれば、全く同じ処理です。
